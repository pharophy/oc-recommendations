## Description

YNK is a hidden-feeling hotel bar that works because it commits to concept changes and mixology framing instead of relying only on low light and a password gimmick.

## What Makes It Unique

Marriott describes YNK as an eclectic speakeasy, while the venue and OpenTable both emphasize an ever-changing menu and a cocktail journey through different regions and eras. That rotating concept makes it more interesting than a static lobby bar pretending to be secret.
